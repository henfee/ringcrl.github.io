Github Pages

什么是Github Pages

Github Pages就是你在GitHub上创建了一个项目,你可以在GitHub上创建无数个项目,但是只能有一个Github Pages,为什么呢?因为这个”特殊”的项目,必须以yourname.github.io来命名,之后呢是以http://yourname.github.io这个网站来访问你所搭建的主页.
